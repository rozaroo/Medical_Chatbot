 # Medical Chatbot

A rule-based medical chatbot developed in Python using spaCy for Natural Language Processing (NLP). The chatbot extracts symptoms from user input, analyzes possible medical conditions using a JSON knowledge base, and provides general recommendations. This project is intended for educational purposes and is not a substitute for professional medical advice.

## Features

- Symptom extraction using spaCy
- Rule-based condition analysis
- Recommendations based on detected conditions
- JSON-based medical knowledge base
- Interactive command-line interface

## Technologies

- Python
- spaCy
- JSON
- Natural Language Processing (NLP)

## Disclaimer

This chatbot is for educational purposes only and should not be used as a replacement for professional medical diagnosis or treatment.
